## File Pattern Inside The `lib` Folder

<img align="left" src="screenshots/folder_structure.png"></img>

```
    lib/
    ├── model/
    │   └── post.dart
    ├── repository/
    │   └── api_repository.dart
    ├── services/
    │   ├── api_service.dart
    │   ├── app_exception.dart
    │   └── base_service.dart
    ├── utils/
    │   ├── api_constants.dart
    │   └── size_config.dart
    ├── view/
    │   ├── widgets/
    │   │   └── post_item.dart
    │   ├── post_details_screen.dart
    │   └── post_list_screen.dart
    ├── view_model/
    │   ├── post_details_view_model.dart
    │   └── post_list_view_model.dart
    └── main.dart
```
